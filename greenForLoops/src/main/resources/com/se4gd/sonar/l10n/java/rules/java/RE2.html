If you are iterating as per the number of item in any collection types and array, calculate the size or length of collection object before hand to avoid calculating the length on every iteration to avoid high energy consumption.

Non-compliant Code Examples


    List<String> listObject = getCollectionData();
    
    for(int i = 0; i < listObject.size(); i++){
        //implementation here
    }
    int[] a = {1,2,3,6};  
    for (int i = 0; i < a.length; i++) {
            //some operations
    }

Compliant Solutions

    List<String> listObject = getCollectionData();
    
    int listSize = listObject.size();
    
    for(int i = 0; i < listSize; i++){
        //implementation here
    }
    
    int[] a = {1,2,3,6};  
    int arrayLength = a.length;
    for (int i = 0; i < arrayLength; i++) {
            //some operations
    }
