Initializing a loop without referring array length consumes 10% less energy than a loop initialized by mentioning array length

Non-compliant Code Examples


    List<String> listObject = getCollectionData();
    
    for(int i = 0; i < listObject.size(); i++){
        //implementation here
    }
    int[] a = {1,2,3,6};  
    for (int i = 0; i < a.length; i++) {
            //some operations
    }

Compliant Solutions

    List<String> listObject = getCollectionData();
    
    int listSize = listObject.size();
    
    for(int i = 0; i < listSize; i++){
        //implementation here
    }
    
    int[] a = {1,2,3,6};  
    int arrayLength = a.length;
    for (int i = 0; i < arrayLength; i++) {
            //some operations
    }
