Avoid using nested loop; instead try to convert nested loop to single loop.

Non-compliant Code Examples

   
    public void test() {
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                // Do some operations
            }
        }
    }

Compliant Solutions

    
    public void test() {
        for (int k = 0; k < n * m; k++) {
            int i = k / m;
            int j = k % m;
            // Do the nested loops operations
        }
    }
