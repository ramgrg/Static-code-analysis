Do not initiate array inside a loop.

Non-compliant Code Examples


   
    for(int i = 0; i < 5; i++){
        int[] a = new int[10]; 
    }
    
    int i = 0;
    while(i < 5){
        int[] b = new int[10]; 
    }

Compliant Solutions

    
    
    int[] a = new int[10]; 
    
    for(int i = 0; i < 5; i++){
        //access the array here...
    }
    
    int i = 0;
    int[] b = new int[10]; 
    while(i < 5){
        //access the array here
    }
