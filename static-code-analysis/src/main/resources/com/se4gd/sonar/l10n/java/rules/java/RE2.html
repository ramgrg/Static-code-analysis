If you are iterating as per the number of item in any collection types, calculate the size or length of collection object before hand to avoid calculating the length on every iteration to avoid high energy consumption.

Non-compliant Code Examples


    List<String> listObject = getCollectionData();
    
    for(int i = 0; i < listObject.size(); i++){
        //....
    }
    
    int[] arraySample = getArray(); 
    for(int i = 0; i < arraySample.length; i++){
        //....
    }

Compliant Solutions

    List<String> listObject = getCollectionData();
    
    int listSize = listObject.size();
    
    for(int i = 0; i < listSize; i++){
        //....
    }
    
    int[] arraySample = getArray(); 
    int arrayLength = arraySample.length;
    
    for(int i = 0; i < arrayLength; i++){
        //....
    }
